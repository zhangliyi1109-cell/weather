## 🔧 技术说明

### Open-Meteo API 调用
```python
API: https://api.open-meteo.com/v1/forecast
参数:
  - latitude: 纬度
  - longitude: 经度
  - daily: 天气代码、最高温、最低温、降水概率、湿度、风向、风速、紫外线、日出日落
  - timezone: Asia/Shanghai
  - forecast_days: 7
```

### WMO 天气代码映射
系统已将 WMO 天气代码转换为中文：
- 0: 晴
- 1-3: 多云/阴
- 45-48: 雾/雾凇
- 51-55: 毛毛雨/小雨/中雨
- 61-65: 小雨/中雨/大雨
- 71-77: 小雪/中雪/大雪
- 80-82: 阵雨/暴雨
- 95-99: 雷雨/雷雨伴冰雹
